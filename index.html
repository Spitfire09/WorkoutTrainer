Data

Sådan importerer du via seed-data.json:
1. Åbn GoogleSheet/seed-data.json — filen indeholder 56 øvelser og 421 log-poster.
2. Send to POST-kald til din Apps Script URL:
  • { "action": "importExercises", "secret": "…", "rows": <exercises-array> }
  • { "action": "importLog", "secret": "…", "rows": <log-array> }
3. Du kan sende disse kald f.eks. med Postman, curl eller fra Apps Script-editoren.
4. Herefter tryk Synkronisér for at trække data ned i appen.
